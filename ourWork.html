GUI - Magic the Gathering Deckbuilder:
- Our GUI is a Magic the Gathering Deckbuilder, the program pulls from the MagicGatherer API in order to let a user search for Magic cards by name and add them to a deck.
- The program was created by Henry Marrone and Kai Glasiek, with an honourable mention to Tai Poole.
- Implementing the API was a considerable challenge. However, utilising it was made much easier by an existing SDK.
- We're hoping to implement ways to save a deck and see additional deck data.


Pictured: a deck list created using the program

Interactive App - Foundry:
- Foundry is a Magic the Gathering Deckbuilder, the program builds off the capabilities of our GUI but with added features. Foundry has all the capabilities of the GUI, however it saves user deck data to a JSON file so decks aren't lost after use of the program. Additionally, we hope to have Foundry find and display relevant data about the user's deck.
- The program is in development by Henry Marrone and Kai Glasiek.
- The program currently has fully working UI and JSON integration.
- Once the current stretch goals are met, we're hoping to have the program write to an Excel Spreadsheet for easy data compilation and analysis.
